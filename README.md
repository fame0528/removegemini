## ğŸ“ How It Works

The application uses a sophisticated reverse alpha blending algorithm to detect and remove Gemini AI watermarks:

1. **Upload**: Select one or more images with Gemini watermarks
2. **Detection**: Automatically detects watermark position and alpha channel
3. **Processing**: Applies reverse blending to reconstruct the original image
4. **Download**: Get your clean images instantly

All processing happens in your browser - no server uploads required!
